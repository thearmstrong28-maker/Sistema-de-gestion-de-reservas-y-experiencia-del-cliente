# Frontend

SPA del sistema, desarrollada con React, Vite y TypeScript.

## Arranque local

```bash
npm install
```

Copiar `frontend/.env.example` a `frontend/.env.local` si todavia no existe.

## Variable principal

```bash
VITE_API_URL=http://localhost:3000
```

## Pantallas principales

- Inicio y autenticacion por rol.
- Registro de la administracion inicial.
- Recepcionista.
- Gerencia.
- Administracion de establecimiento.
- Clientes, reservas, disponibilidad, lista de espera y reportes.

## Validacion

```bash
npm run lint
```
